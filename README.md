📊 Controle de Aferições

Sistema desenvolvido para gerenciar e controlar aferições de equipamentos, garantindo organização, rastreabilidade e confiabilidade nos dados.

🚀 Sobre o Projeto

O Controle de Aferições tem como objetivo facilitar o registro e acompanhamento de aferições realizadas em equipamentos, permitindo um controle mais eficiente e seguro das informações.

🧩 Funcionalidades
📌 Cadastro de aferições
📌 Registro de equipamentos
📌 Armazenamento de dados históricos
📌 Organização das informações para consulta
📌 Controle simples e eficiente
